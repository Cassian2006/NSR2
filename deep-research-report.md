# 极地 NSR 避险航线规划平台升级计划（清理版）

## 1. 目标定位
在现有“可演示系统”基础上，升级为“可复现、可解释、可对比、可合规”的研究级与工程级平台。

## 2. 四条主线（总览）
1. 数据与版本化底座
2. 风险规避与鲁棒规划
3. 时序数字孪生与增量重规划
4. 评测基准与合规报告

## 3. 主线说明

### 主线1：数据与版本化底座
- 统一数据时间步、投影、命名与版本号。
- 建立数据清单与产物追踪链：数据版本 -> 模型版本 -> 规划版本 -> 报告版本。
- 固化可复现实验入口（同参数可重复得到同结论）。

### 主线2：风险规避与鲁棒规划
- 从“单一最短路”升级到“风险约束规划”。
- 支持保守/平衡/激进多策略路线并行输出。
- 引入不确定性校准后的风险场，避免直接使用未校准置信度。

### 主线3：时序数字孪生与增量重规划
- 按时间步更新环境场，触发 D* Lite 增量重规划。
- 输出重规划事件日志与轨迹演化回放。
- 对比 A* / D* Lite / Any-angle / Hybrid A* 在动态场景中的收益边界。

### 主线4：评测基准与合规报告
- 建立固定场景基准集与统一指标矩阵。
- 增强 AIS 回测：轨迹相似度 + 显著性检验。
- 形成结构化合规报告（风险摘要、规则提示、适用边界）。

## 4. 推荐里程碑（12周）
- W1-W2：数据与版本化骨架
- W3-W4：评测基准框架 v1
- W5-W6：风险规避规划 MVP
- W7-W8：动态孪生回放 MVP
- W9-W10：合规与报告模板 v1
- W11-W12：竞赛/论文交付包

## 5. 交付物清单
- 可复现实验脚本与运行说明
- 路线规划多策略接口与前端交互
- 动态重规划日志与回放页面
- 基准评测报告（含统计检验）
- 合规与解释报告（导出）

## 6. 风险与边界
- 不确定性未经校准会误导风险判断。
- 动态数据链路不稳定会影响 latest 体验。
- 合规模块应定位为“决策支持”，不是自动驾驶替代。
